# Добро пожаловать в Settlers

SETTLERS превращает ваше реальное передвижение в стратегическую игру по контролю территорий. Захватывайте гексагональные территории, посещая локации, присоединяйтесь к кланам и соревнуйтесь за влияние на глобальной карте — всё через Telegram.
